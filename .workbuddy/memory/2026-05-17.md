# 2026-05-17 工作日志

## Bug修复：攻击漂移 + BO长按失效（第2轮）

### 第1轮（未完全解决）
- **攻击漂移**：添加 `velocity = Vector2.ZERO` 每帧清零，设置 `motion_mode = MOTION_MODE_FLOATING`
- **BO长按失效**：地面普攻改为计时器驱动（`attack_timer + attack_duration=0.4s`），不再依赖 `animation_finished` 回调
- 攻击动画长度 0.2s → 0.5s

### 第2轮（加强修复）
- **攻击漂移根治**：添加 `attack_start_pos: Vector2`，在 `_start_attack()` 和 `_start_air_attack()` 保存初始位置，`_tick_attack()` 和 `_tick_air_attack()` 每帧强制 `global_position.x = attack_start_pos.x`，从底层彻底杜绝任何水平位移
- **BO双重检测**：`_tick_attack()` 中同时检查 `is_attack_held` 脚本变量和 `Input.is_action_pressed("attack")` 原始输入状态，防止 Unity 风格的按键事件丢失
- **BO动画长度修复**：`player.tscn` 中 BO 动画添加显式 `length = 1.0`，防止无 length 属性导致动画瞬间完成
